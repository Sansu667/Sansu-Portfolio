About me

I am a philosophy student at the National University of Colombia, where I explore the complexities of critical thinking and reflection. My passion for programming has led me to develop projects in Python and Javascript, tools I use to bring my ideas to life. I have collaborated with an educational institution with the goal of preserving the memory of a neighborhood, creating programs that foster connection and remembrance of our local history. Welcome!

Programming Projects

Batatabit

Batatabit is an interactive website that allows users to compare real-time prices of various cryptocurrencies. This project, developed using HTML, CSS, and JavaScript, focuses on providing reliable and up-to-date information about the cryptocurrency market.

With its intuitive design, users can easily access relevant data, facilitating informed decision-making in their investments. Batatabit combines functionality and aesthetics, offering a valuable tool for cryptocurrency enthusiasts and traders.

Space War

Space War is a game developed with Python, and with the help of the tools provided by the Pygame library. The game is 2D arcade style.

It consists of defeating the invaders and not letting them reach us. Other libraries such as Math, Random and Io were used. This game was made with the help of maestro Federico Garay.

Facturation System

This facturation system is made in Python using the Tkinter library. It consists of a system where you have a menu and you can select which dish/drink/dessert you want to have and how many units there are.

From this it makes the total calculation and allows you to create a receipt, save the receipt and reset the system.

Secret Game

Secret Game is an entertaining interactive game developed in HTML, CSS, and JavaScript, where players must guess a randomly generated secret number. As they make their guesses, they will receive hints to help them get closer to the correct answer.

With an attractive design and simple gameplay, this game is perfect for players of all ages looking for a fun and educational challenge. The combination of logic and strategy provides an engaging experience that stimulates the mind and enhances problem-solving skills.

Una semana de memorias

Una semana de memorias es una colección de relatos donde en cada día se va narrando una historia diferente. Cada historia tiene un tono diferente y un estilo único, pero todas ellas comparten un hilo conductor que las une. A través de estas historias, el lector podrá sumergirse en un mundo de emociones y reflexiones, donde la imaginación y la realidad se entrelazan para crear un universo único y fascinante.
