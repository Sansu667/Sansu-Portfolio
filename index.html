About Me

I’m Santiago Suarez, a Software Developer and Data Analyst driven by curiosity, creativity, and impact. With a background in philosophy, statistics, and web development, I build elegant, responsive web applications and extract meaning from data using modern tools and frameworks.
As a freelancer, I’ve built and maintained websites using HTML, CSS, JavaScript, and Django, combining front-end design with backend logic. I also develop data solutions with Python, Power BI, and Excel, applying statistical thinking to solve real-world problems. I continuously expand my toolkit with courses from Platzi and formal studies at Colombia’s National University and SENA.

I value technology with purpose. My project “Cuita Wedacanictaco” helps preserve the cultural memory of Villavicencio, blending code with community. I also created a responsive Text Encryptor/Decryptor in JavaScript, showcasing practical logic and clean UI design.
I’m fluent in Spanish, English, and Italian — ready to collaborate across borders and cultures. I’m currently open to new opportunities in web development, software engineering, and data analysis where I can learn, contribute, and create meaningful solutions.
